To execute our project, open a terminal and navigate to this folder.
Now type "gradle assemble". (The build should be successful)
The complete jar is now in the group6/bin directory.
Go to this directory.
Now execute the command "Java -jar swp2025_group6.jar --server ServerIP --port Portnummer",
using the servers IP and Port and our client will start and try to connect to the server.
